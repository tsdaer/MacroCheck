Reports C/C++ macro definitions that lack corresponding #undef directives.

This inspection helps maintain clean macro usage by ensuring that every #define macro has a matching #undef directive. This is particularly important for header files to prevent macro pollution across different compilation units.

Example of problematic code: 


#define MAX_SIZE 100
// ... code using MAX_SIZE ...
// Missing: #undef MAX_SIZE

Example of correct code: 


#define MAX_SIZE 100
// ... code using MAX_SIZE ...
#undef MAX_SIZE

Text after this comment will only be shown in the settings of the inspection.

To configure this inspection, go to Settings | Editor | Inspections | Macro Definition Check
